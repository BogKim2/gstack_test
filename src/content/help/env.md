# 환경 변수

상세 목록과 예시는 프로젝트 루트 **`.env.example`** 을 열어 참고하세요.

| 구분 | 예시 키 |
|------|---------|
| Auth.js | `NEXTAUTH_URL`, `NEXTAUTH_SECRET` |
| Google | `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET` |
| 선택 소셜 | `KAKAO_*`, `NAVER_*` |
| DB | `DATABASE_URL` |
| 암호화 | `SECRET_KEY`, 선택 `OAUTH_TOKEN_SECRET` |
| LLM | `OPENAI_API_KEY`, LM Studio 관련 |
