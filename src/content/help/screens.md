# 화면 (App Router)

| URL | 소스 파일 | 역할 |
|-----|-----------|------|
| `/` | `src/app/page.tsx` | 랜딩 |
| `/auth/signin` | `src/app/auth/signin/page.tsx` | 로그인 |
| `/dashboard` | `src/app/dashboard/page.tsx` | 대시보드·브리핑 |
| `/settings` | `src/app/settings/page.tsx` | 설정 |
| `/settings/connect-google` | `src/app/settings/connect-google/page.tsx` | Google 연결 |

전역 레이아웃: `src/app/layout.tsx`
